CSS

Если я установлю сам какую-то ширину блочного элемента, например параграфа, то тогда могу установить этот элемент по центру страницы. Элемент теперь станет короче контейнера в котором находится, например тега <body> или тега <div>. И теперь блочный элемент можно двигать влево или вправо по горизонтали. Можно написать код примерно так: 

p {
   margin: auto;
}

Селекторы атрибутов

Можно получить элемент на основании того, что его атрибут начинается с определённого текста. Используется такой вариант тег[атрибут^="наш текст"].

Также можно получить элемент на основании того, что указанный текст есть где-то в значении атрибута. В начале, в середине или в конце атрибута. Тогда используем такой вариант [атрибут*="наш текст"].
